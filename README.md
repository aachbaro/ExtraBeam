# ExtraBeam

## Description

Plateforme web permettant de présenter son CV, ses disponibilités, de recevoir et gérer des propositions de mission, ainsi que ses factures.

## Objectif

Produire un SaaS complet avec une structure modulaire, apprendre à utiliser Stripe et créer un système d’abonnement.

## Technologies
- Vue 3 / Tailwind
- NestJs
- Supabase
- Render
- Brevo
- Docker
- Typescript
- Stripe
- Vercel

## Fonctionnalités

- Agenda avec slots de disponibilités
- Comptes entreprise / client
- Demandes de mission
- Génération de factures
- Facturation via compte Stripe
- Emails de notification

## Etat du projet

Une version du site est déjà accessible à l’adresse extrabeam.app, hébergée gratuitement sur Render pour le backend et Vercel pour le frontend.
Je suis actuellement en train de mettre en place le système d’abonnement dans une autre branche.
Il restera des ajustements visuels et fonctionnels sur plusieurs aspects du site.

## Contexte

Projet concret et complet à but éducatif, pouvant évoluer vers un projet commercial si les tests personnels sont concluants.
